#### Flow
1. To set up the project do npm create vite@latest my-project 
2. Cd to the project file and install tailwind, then delete vite stuff
3. Keep index.css and type in it @import "tailwindcss";
4. In vite.config.ts put tailwindcss() in the plugins section
